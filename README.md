# StudentSync
For my mobile app development course capstone project, I will build StudentSync, a mobile app designed to help college students keep track of what is happening on campus. Right now, if you want to find out about an event, a scholarship, or even just where the tutoring center is, you end up digging through emails, the non-updated website, or hoping you happened to see a flyer.  

The purpose of this app is to put everything in one place. Students should be able to open one app, search for something, and find out whether it is a FAFSA workshop, a coding club meeting, or advising hours.
